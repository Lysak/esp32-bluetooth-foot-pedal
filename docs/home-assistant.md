# Home Assistant

The ESP32 is exposed through the standard ESPHome integration.

Home Assistant is for visibility, diagnostics, counters, and maintenance. The key press stays local on the ESP32 and does not depend on Home Assistant automations.

Target version 1 entities:

- pedal state
- BLE connected state
- key held state
- press counters
- hold duration
- persistence status
- current binding
- uptime
- Wi-Fi RSSI
- restart
- reset statistics
